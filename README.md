### Mysqlite -- Nodejs
Login Authentication included with password encrption

### setup app using package.json
1. Clone the repo
2. `npm install`
3. 	run `nodemon server.js` to start
4. visit `localhost:8000` on your browser


### Api Route
1. Register
```bash
	> /api/register
```

2. Register as Admin
```bash
	> /api/register-admin
```

3. Get all users
```bash
	> /api/users
```

4. login
```bash
	> /api/login
```
